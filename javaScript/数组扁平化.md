### 数组扁平化 ###

var arr = [1, [2, [3, 4]]];
对于数组嵌套数组的结构，我们如何实现将它变为 [1,2,3,4]的结构呢？

首先，我们可能想到的是递归

	function flattening(arr){
	
		var newArr = [];

		for(var i=0;i<arr.length;i++){
			
			//判断该项是不是数组，如果是数组就递归
			if(Array.isArray(arr[i])){
				newArr= newArr.concat(flattening(arr[i]))
			}else{

				newArr.push(arr[i]);
			}
		
		}
	 	return newArr;
	}

除此之外呢？reduce也是可以的
